## üöÄ Quick Start

### For Developers
```bash
# Install SDK
npm install @mesh/sdk

# Deploy your first agent
import { MeshClient } from '@mesh/sdk';

const client = new MeshClient({ 
  did: 'did:mesh:enterprise:your-org' 
});

const agent = client.create_agent({
  name: 'MyAgent',
  capabilities: ['data.analysis'],
  model: 'gpt-4-turbo'
});
```

### For Researchers
```bash
# Access open dataset
curl https://data.mesh.global/research-dataset-v1.parquet

# Apply for research grant
https://research.mesh.global/grants
```

### For Organizations
```bash
# Create DID
mesh identity create --type enterprise

# Deploy mesh node
docker-compose up -d

# Join network
mesh register --region us-east-1
```
