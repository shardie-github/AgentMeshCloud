## Asset Specifications

### Logo Requirements
- Format: PNG with transparency
- Dimensions: 512×512px minimum
- Color Space: RGB
- Max File Size: 1MB

### Screenshot Requirements
- Format: PNG or JPG
- Dimensions: 1920×1080px (16:9 aspect ratio)
- High quality, no compression artifacts
- Include realistic data (use demo mode)
- Annotate with arrows/highlights where helpful

### Video Requirements
- Format: MP4 (H.264)
- Duration: 60-90 seconds
- Resolution: 1080p minimum
- Frame Rate: 30fps
- Audio: Clear voiceover + background music
- Captions: Include English subtitles
